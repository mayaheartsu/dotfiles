## acquire config files

```
git clone https://github.com/mayaheartsu/dotfiles
cd dotfiles
cp -r ./config/* ~/.config/
```

## install needed packages
run the extremely basic-ass install script to acquire the appropriate packages
```
cd dotfiles
./install.sh
```
